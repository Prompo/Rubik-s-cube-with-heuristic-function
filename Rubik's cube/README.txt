Για να εκτελεστεί το πρόγραμμα:
1. Ανοίξτε το Command Prompt
2. Μέσω της εντολής "cd", ορίστε το path που βρίσκεται ο φάκελος της εργασίας
	Πρέπει να είναι της μορφής "cd _:\...\3190138_3190179"
3. Τρέξτε την εντολή: cd CuBot
3. Τρέξτε την εντολή: javac Main.java
4. Τρέξτε την εντολή: java Main
